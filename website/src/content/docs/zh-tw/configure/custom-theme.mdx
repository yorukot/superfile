### 使用現有主題

你可以輸入下列指令來設定：

[點我了解 CONFIG_PATH 在哪裡](/zh-tw/configure/config-file-path#config)

```bash
$EDITOR CONFIG_PATH
```

你可以先到[主題列表](/zh-tw/list/theme-list)找一個喜歡的主題（如果沒有喜歡的，也可以自己做一個！）

找到喜歡的主題後，把它複製並貼到 config_path 檔案中的 theme。

```diff
- theme = 'catppuccin'
+ theme = 'theme_name_you_like'
```

### 建立你自己的主題

[點我了解 THEME_DIRECTORY 在哪裡](/zh-tw/configure/config-file-path#config)

如果你想自訂自己的主題，可以前往 `THEME_DIRECTORY/YOUR_THEME_NAME.toml`，並將現有主題的 `toml` 複製到你自己的主題檔中。

別忘了將 `config.toml` 中的 `theme` 變數改成你的主題名稱。

[如果你對自己的主題很滿意，不妨把它加入預設主題列表！](/zh-tw/contribute/how-to-contribute)
